7 | View test results in a report

Do one of the following to access the Perfecto report:

  • Use the URL to the report provided during driver creation through the testGridReportUrl capability. 

  • In the Perfecto UI, go to Test Analysis > Report Library and find your report. The default name is RemoteWebDriver.

    Click to view sample report

The report has a generic name, RemoteWebDriver. You can customize the name and enrich the report content by implementing the Perfecto Reporting framework through the Reporting SDK.
