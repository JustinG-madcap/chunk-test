First steps in the public cloud

Welcome to Perfecto's Public Cloud! This article helps you get started. It walks you through the process of logging in, opening a device, and adding users, org groups, and roles.

Groups are useful for organizing users within Perfecto, for example by manager, project, or other criteria that work well for your team. Groups do not impact a user's access privileges within Perfecto. Those are controlled by roles.

Roles allow an administrator to determine the level of access a team member has within Perfecto. The roles available to your team are determined by the Public Cloud Package you have purchased. Standard roles might include:

  • Administrator: The Lab Admin with full privileges for your Perfecto Public Cloud. This would be a good role for the team lead or manager that is responsible for overall access to the Perfecto Public Cloud environment
  • Handset Viewer: A typical Perfecto user. This role allows a user to perform manual tests, view and stop executions, and view reports.

For more detailed information on these administrative tasks, see Manage users. For more information on manual testing on devices in the cloud, see Manual testing.

1 | Log in

  • After completing the sign-up process, go to https://mobilecloud.app.perfectomobile.com and log in using the username and password provided in the welcome email you received from Perfecto.

2 | Select a device

  1. On the landing page, click OPEN DEVICE.

  2. In the Manual Testing view, find a device you want to work with.
    By default, the Manual Testing view opens in tile view , but you can switch to a row view  if you prefer.
     
    If you have been here before, you can select Recent in the right pane and choose a device you worked with previously. At any time, you can sort by manufacturer or use the search box at the top right if there is a specific device you would like to use.
  3. Select the device to display the details pane on the right. This pane provides details on the device, such as availability, OS version, device ID, or location, and available capabilities.
  4. On the device tile or in the selected device row, click OPEN.
    Now you can begin testing on the device. For information on the testing functionality available in the public cloud, see Public Cloud updates.
    At any time, you can add additional users to your Perfecto Public Cloud. For details, see step 3.

3 | Add members to your team

  1. Click the menu at the top and select Assets and Setup.
  2. In the Assets and Setup view, click Admin to open the Users tab.

  3. On the Users tab, above the table, click Add user .

  4. In the Add user form, provide user information and assign a username and password. 

    For information on adding groups, see step 4.
  5. To add additional users, turn on the Add one after another option.
  6. Then click Add.
  7. (Optional) To edit user information or delete a user:
    1. Select the user in the table.
    2. In the toolbar above the table, click More actions and select the required option.

4 | Add org groups and assign members to groups

  1. To add a group to your cloud, in the Org groups pane on the left, click Add .

  2. In the Add Org group form, provide the following information:
    • Group name, for example My web team
      The group name cannot exceed 25 characters.
    • Description, for example Used by my web team
      The description cannot exceed 50 characters.
  3. Add users to the group using the Search field. You can enter a username or an email address.
  4. To add additional org groups, turn on the Add another when done option. You can also come back to adding users to a group later, whenever the need arises.
  5. Click ADD.
  6. (Optional) To edit a group:
    1. In the right pane, select the org group you want to edit.
    2. Click the pencil icon  on the right.
    3. In the Edit Org group form, make the required changes and click SAVE.
  7. (Optional) To delete a group:
    1. In the right pane, select the org group you want to delete.
    2. Click the trash can icon  on the right.
    3. When prompted for confirmation, click YES.

5 | Add roles

  1. To add a role to a user, select a user in the table.
  2. In the details pane on the right, on the ROLES tab, turn on the role or roles you want to assign to the user.
    You can change a user's role settings at any time.
