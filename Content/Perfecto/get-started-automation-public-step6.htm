6 | Finalize and run your script

  1. Verify that your script includes all relevant details to run with Perfecto:
    • Perfecto cloud URL
    • Security token
    • Capabilities
    • App file locator
  2. If you test a native or hybrid app, update the bundleId (iOS) or appPackage (for Android) capability.
    For more information on capabilities, see Supported Appium capabilities.
  3. Run the script.
