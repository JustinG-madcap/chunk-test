2020 releases

Release 20.1

Learn about the new change password and application management options and other enhancements.

Release 20.2

Learn about the new Perfecto Status page, our integration with Katalon, session sharing for web, app management, support for secure screens on Android 10, and other updates.

Release 20.3

Learn about support for the latest Samsung devices, Perfecto integration with NeoLoad, the biometric API for Android, the new device dashboard, the Set Location filter hidden fields in iOS 13.x devices, concurrent device sessions, session timeout after inactivity, and other platform and device updates.

Release 20.4

Learn about accessibility testing, new options for fingerprint/face ID injections, network virtualization, and initiating a call or SMS to the open device, a new way to create, execute, and schedule maintenance scripts, and support for the latest Android and iOS versions.

Release 20.5

Learn about Perfecto integration with Detox, new Manual testing features such as the extended device toolbar, HAR file recording for desktop web testing, image and audio injection, and device log retrieval, a new API to retrieve license information, a new parameter to turn off automatic timeout, as well as support for secure popups on iOS 13.x devices, AndroidX support for the Perfecto Gradle plugin for Espresso, and platform support updates.

Release 20.6

Learn about the Spring 2020 Mobile & Web Test Coverage Index, integration with Ivy, support for the latest platforms and devices, and more.

Release 20.7

Learn about visual analysis, new integrations with Test Modeller, 21, Axe, and Tosca (for native mobile apps), audio connection and SIM Card availability at a glance, support for the latest platforms and devices, and more.

Release 20.8

Learn about repository management, Perfecto ingeration with Botium, device and app vitals, testing on emulators, and support for the latest browsers and platforms.

Release 20.9

Learn about support for iOS 14 Developer Beta, virtual devices, User Management and DevTunnel features in the Perfecto UI, the Perfecto integration with Jest, as well as newly supported browser versions and device platforms.

Release 20.10

Learn about reservations, file transfer, support of WebdriverIO for Desktop Web, as well as platform and browser support updates.

Release 20.11

Learn about support for the latest Samsung devices, the new Jenkins plugin, visual analysis enhancements, platform support updates, and devices now considered deprecated.

Release 20.12

Learn about reservations in manual testing, visual analysis enhancements, testing on foldable phones, new supported devices, and platform and browser support updates.

Release 20.13

Learn about the Perfecto the new Object Spy, support for APKS files, Visual Analysis enhancements, support for Selenium 4 ALPHA, the new Android API level 29, new supported platforms, and browser support updates.
