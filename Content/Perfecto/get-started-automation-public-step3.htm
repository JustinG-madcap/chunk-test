3 | Generate your security token

  1. Go to https://testingcloud.app.perfectomobile.com and log in using the username and password provided in the welcome email you received from Perfecto.

  2. On the landing page, click your user name and select My security token.

  3. In the My security token form, click Generate Security Token.

  4. Click Copy to clipboard.

  5. Click Close.
  6. Paste the security token into your script.
