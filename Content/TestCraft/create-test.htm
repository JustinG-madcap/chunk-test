Create a spectest

Creating a new spectest only takes you a couple of clicks. Then you can continue to edit your spectest either on the canvas or on the fly.

To create a spectest:

  1. Access the SpecTest view.

  2. In the SpecTest view, navigate to the folder in which you would like to create the spec.

  3. Above the spectest list, click Add spectest.

  4. In the Add SpecTest form, do the following:

    1. Enter a spectest name.

    2. Select whether you want to build the spectest on the fly (default) or on the canvas.

    3. Click ADD.
