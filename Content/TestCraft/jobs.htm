Test suites

The Test SuitesJobs view provides a list of all existing test suitesjobs, along with their creation date and description. A test suitesjobs is a group of tests that runs together. You may be more familiar with the term jobs. In the Test SuitesJobs view, you can:

  • Create test suitesjobs

  • Edit test suitesjobs

  • Rename test suitesjobs

  • Delete test suitesjobs

  • Play test suitesjobs

  • View test suitesjobs's executions

To view all test suitesjobs:

Click the menu icon at the top left and select Test SuitesJobs. The Test SuitesJobs view opens.
