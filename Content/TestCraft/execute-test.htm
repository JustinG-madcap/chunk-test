Execute a test in 3 clicks

The first step toward running your test only takes 3 clicks. The navigation elements are already included in your spec, so you can start right away.

To run your first test:

  1. At the top right, click Run.

  2. In the Run Test form, if you prefer not to pause the test at the last step, clear the Add breakpoint at the end of the test checkbox. This option is selected by default.

  3. Click RUN.
