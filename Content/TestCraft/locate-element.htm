Locate a missing element

When running or re-running a test, you might encounter a missing binding element. In this case, the test stops and an orange bar blinks across the browser pane, alerting you that the test element could not be located in the browser.

If a break in the test occurs, you can change it while your test is still running, on the fly. This is a unique feature of our platform.

To fix a missing binding:

  1. In the browser window, identify the element that corresponds to the test step and click it.

  2. From the quick action bar, select the checkmark icon to confirm the element.

  3. The test continues to run.
