Create a test suite job

A test suite job in is a group of tests that run together. You may be more familiar with the term job.

This article explains how to create a test suite job.

The tests in a test suite job should be independent from each other, meaning that each testspec should be able to run on its own. This will pay off later when you need to maintain your application.

To create a test suite job job:

  1. Access the Test Suites Jobs view.

  2. In the Test Suites Jobs view, at the top right, click Add Test Suite Job.

  3. In the Add Test Suite Job form, enter a name and, optionally, a description.

  4. Click ADD.
